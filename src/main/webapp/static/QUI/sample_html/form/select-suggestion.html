说明 提示下拉框点击右侧箭头按钮展开下拉列表，点击文本框可以进行输入过滤。
默认有输入验证功能，即输入的文本在列表里面不存在时，会自动清除。此功能可禁用，这时可以输入任意文本。
相比单选下拉框组件，除了有输入提示功能外，还有选项标签式分组，数据列表分页，键盘操作进行翻页和选中等额外功能（键盘操作在点击文本框展开列表时有效，点击按钮时无效）。缺少联动，动态修改选项功能。
