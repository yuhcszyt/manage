2010年职称评审工作已经开始，本年度参与评审人员任职时间计算至2010年8月。工程师及高级工程师均需参加答辩，通过全国计算机技术与软件专业技术资格（水平）考试的人员不参加答辩，可以比正常晋升职称提前一年申报。报名费100元/每人。
外聘人员通过全国计算机技术与软件专业技术资格（水平）的可以申报评审，其它情况不予受理。
申报材料截止日期为2010年9月3日，过期不再受理。

一、高级工程师申报条件
1、博士毕业并获得博士学位后，获得工程师资格并在工程师岗位工作两年以上。
2、硕士毕业并获得硕士学位后，获得工程师资格并在工程师岗位工作四年以上。
3、本科毕业后，获得工程师资格并在工程师岗位工作五年以上。

二、工程师申报条件
1、本科：助理工程师满四年。
2、硕士：助理工程师满三年。
注：通过全国计算机技术与软件专业技术资格（水平）考试的人员以上年限均减一年。
